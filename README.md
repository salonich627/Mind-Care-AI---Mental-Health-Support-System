# MindCare AI – A Mental Health Support System

MindCare AI is an AI-powered mental health support web application designed to provide emotional support and mental wellness assistance through an interactive and user-friendly interface.

The system helps users manage stress, anxiety, and emotional well-being using AI-based conversations, mood tracking, calming activities, and self-help features.

## Features
- AI-powered mental health chat support
- Mood tracking system
- Interactive and responsive user interface
- Mental wellness activities
- Self-help quizzes and exercises
- Calming music and stress relief support
- Secure and smooth user experience

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Groq API

## Project Objective
The main objective of this project is to create a smart and supportive platform where users can openly interact, track emotions, and improve mental wellness through AI assistance and engaging activities.
